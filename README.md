# The Crypto Wallet
The crypto wallet mobile app. 

## getting started

### Pre-requisits

- Expo
- Nodejs
- Simulator Android or Ios

### Running apps

- clone the app
- run 
```sh
$ yarn install
# npm run install
```

- starting the app

```sh
$ expo start
```
